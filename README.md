ASnake
======

Snake game implemented in x86-64 Assembly. The graphics is 
done using SDL routines.

![ASnake](demo.gif?raw=true "ASnake")

**dependencies:**
```bash
libsdl
```
**compile and link:**
```bash
make
```
**run:**
```bash
./a.out
```
