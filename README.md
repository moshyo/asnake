ASnake
======

A simple snake game implemented in x86-64 Assembly. The graphics is 
done using SDL routines therefore libsdl is needed when compiling.

![ASnake](demo.gif?raw=true "ASnake")

compile with:
```bash
make
```

and run with:
```bash
./a.out
```
